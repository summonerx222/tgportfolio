Telegram Bot Landing

Что внутри:
- index.html
- style.css
- script.js

Как запустить:
1. Открой index.html локально в браузере
или
2. Залей все файлы в корень GitHub-репозитория
3. Подключи репозиторий к Cloudflare Pages

Что нужно сделать перед публикацией:
- Открой script.js
- Найди строку:
  const WEBHOOK_URL = "https://example.com/webhook";
- Замени URL на свой webhook:
  - webhook.site для теста
  - webhook из n8n / Make / backend для реальной демонстрации

Важно:
Все файлы лежат в одной папке, без подпапок.
